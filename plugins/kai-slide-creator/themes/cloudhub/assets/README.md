# Cloudhub Assets

本目录存放云之家品牌风格的图片资源。

## 文件用途速查表

| 文件名 | 用途 | 页面位置 |
|--------|------|----------|
| `cloudhub_white.png` | 云之家 Logo（白底用） | 首页左上角、章节页右上角 |
| `cloudhub_blue.png` | 云之家 Logo（蓝底用） | 目录页/内容页/尾页左上角 |
| `cloudhub_homepage.png` | 首页全屏背景图 | 首页 |
| `Cloudhub_slogan2025.png` | Slogan 图 | 首页右上角、尾页左侧 |
| `cloudhub_endpage.png` | 尾页全屏背景图 | 尾页 |
| `cloudhub_thanks.png` | 感谢图 | 尾页右侧 |

## 复用 Kingdee 的背景图片

- `kingdee_catalogue.png` — 目录页全屏背景
- `kingdee_chapter.png` — 章节页全屏背景

## Logo 规范

- 尺寸建议：高度 48-54px
- 格式：PNG 透明背景
- 白底用 `cloudhub_white.png`，蓝底用 `cloudhub_blue.png`
