# ASCII Stream — Style Reference
# 字符流 — 风格参考文档

Live typographic streams meet the slide deck. Inspired by ASCILINE, the Matrix shell, and old serial terminals — every surface breathes characters. Deep ink background, mono glyphs raining behind the content, accent in phosphor green or hot amber.

字符流动美学。灵感来自 ASCILINE、《黑客帝国》终端和老式串口屏——每一个面都在"呼吸"字符。深墨底色，单字符雨在内容背后流动，强调色用磷光绿或热橙。

## Colors / 配色

```css
:root {
    --bg:           #07090C;   /* near-black ink / 近黑墨色 */
    --bg-secondary: #0E1218;   /* card surface / 卡片底 */

    --text-muted:   #6B7785;   /* dimmed glyph / 暗化字符 */
    --accent:       #5BFF8E;   /* phosphor green — primary accent / 磷光绿 */
    --accent-warm:  #FFB347;   /* amber — secondary accent / 热橙副强调 */
    --accent-dim:   rgba(91,255,142,0.15);
    --rule:         rgba(230,237,243,0.08);
}
```

**Forbidden / 禁止使用：** drop shadows、gradients with multiple hues、serif fonts、rounded corners > 4px、emoji、decorative icons. 单一磷光色调，禁止彩虹/糖果色搭配。

## Background — Character Rain (Signature Element) / 字符雨（签名视觉）

A canvas behind every slide rains random ASCII glyphs at low opacity. Generated by JS at runtime; no images, no external assets.

每张幻灯片背后都有一层 canvas 在下"字符雨"，由 JS 在运行时生成，零外部资源。

```css
#ascii-rain {
    position: fixed;
    inset: 0;
    z-index: 0;
    pointer-events: none;
    opacity: 0.22;          /* keep faint so content stays readable / 保持微弱以免干扰内容 */
    mix-blend-mode: screen;
}

body { background: var(--bg); }

.slide {
    position: relative;
    z-index: 1;             /* sit above the rain / 浮在雨层之上 */
}

@media (prefers-reduced-motion: reduce) {
    #ascii-rain { display: none; }
}
```

The canvas is generated by this code (drop into `<script>` near the top of body):

```js
(function asciiRain() {
    if (matchMedia('(prefers-reduced-motion: reduce)').matches) return;
    const canvas = document.getElementById('ascii-rain');
    if (!canvas) return;
    const ctx = canvas.getContext('2d');
    const glyphs = '01░▒▓█▌▐■▪╳╱╲┃━┓┛┏┗01<>{}[];=+/\\#@$%&'.split('');
    let cols = 0, rows = 0, drops = [];
    const fontSize = 14;

    function resize() {
        canvas.width  = innerWidth;
        canvas.height = innerHeight;
        cols = Math.ceil(canvas.width / fontSize);
        drops = Array.from({ length: cols }, () => Math.random() * canvas.height / fontSize);
    }
    resize();
    addEventListener('resize', resize);

    let last = 0;
    function frame(t) {
        if (t - last > 60) {                       // ~16fps for low CPU
            ctx.fillStyle = 'rgba(7,9,12,0.18)';   // soft trail fade
            ctx.fillRect(0, 0, canvas.width, canvas.height);
            ctx.fillStyle = '#5BFF8E';
            ctx.font = fontSize + 'px JetBrains Mono, monospace';
            for (let i = 0; i < cols; i++) {
                const ch = glyphs[(Math.random() * glyphs.length) | 0];
                ctx.fillText(ch, i * fontSize, drops[i] * fontSize);
                if (drops[i] * fontSize > canvas.height && Math.random() > 0.975) drops[i] = 0;
                drops[i]++;
            }
            last = t;
        }
        requestAnimationFrame(frame);
    }
    requestAnimationFrame(frame);
})();
```

## Animation / 动画

Three signature character-level animations:

1. **Typewriter reveal** — characters print left-to-right with steps() timing
2. **Glyph scramble** — numbers and short labels flicker random glyphs before settling
3. **Standard fade-up** — for body content and cards

```css
/* Typewriter — for titles and headings
   打字机效果——用于标题 */
.as-typewriter {
    display: inline-block;
    overflow: hidden;
    white-space: nowrap;
    border-right: 0.6ch solid var(--accent);
    width: 0;
}
.slide.visible .as-typewriter {
    animation:
        as-type 1.4s steps(40, end) 0.2s forwards,
        as-blink 0.7s steps(2, start) infinite;
}
@keyframes as-type  { to { width: 100%; } }
@keyframes as-blink { 50% { border-color: transparent; } }

/* Standard reveal — fade up
   标准淡入 */
.as-reveal {
    opacity: 0;
    transform: translateY(14px);
    transition: opacity 0.55s ease, transform 0.55s ease;
}
.slide.visible .as-reveal              { opacity: 1; transform: translateY(0); }
.slide.visible .as-reveal:nth-child(1) { transition-delay: 0.1s; }
.slide.visible .as-reveal:nth-child(2) { transition-delay: 0.25s; }
.slide.visible .as-reveal:nth-child(3) { transition-delay: 0.40s; }
.slide.visible .as-reveal:nth-child(4) { transition-delay: 0.55s; }
.slide.visible .as-reveal:nth-child(5) { transition-delay: 0.70s; }

/* Glyph scramble — JS-driven, see snippet below
   字符乱码定身——由 JS 驱动 */
.as-scramble { font-variant-numeric: tabular-nums; }

@media (prefers-reduced-motion: reduce) {
    .as-typewriter { animation: none; width: 100%; border-right: none; }
    .as-reveal     { transition: none; opacity: 1; transform: none; }
}
```

The scramble animation runs once per slide entry. Drop this in the same `<script>` block:

```js
function scrambleOnEnter() {
    const glyphs = '!<>-_\\/[]{}—=+*^?#01';
    document.querySelectorAll('.slide').forEach(slide => {
        const targets = slide.querySelectorAll('.as-scramble');
        const observer = new IntersectionObserver(es => es.forEach(e => {
            if (!e.isIntersecting) return;
            targets.forEach(el => {
                const final = el.dataset.text || el.textContent;
                el.dataset.text = final;
                let frame = 0, total = 18;
                const tick = () => {
                    let out = '';
                    for (let i = 0; i < final.length; i++) {
                        out += i < (frame / total) * final.length
                            ? final[i]
                            : glyphs[(Math.random() * glyphs.length) | 0];
                    }
                    el.textContent = out;
                    if (frame++ < total) requestAnimationFrame(tick);
                    else el.textContent = final;
                };
                tick();
            });
        }), { threshold: 0.5 });
        observer.observe(slide);
    });
}
scrambleOnEnter();
```

## Signature Elements / 签名视觉

1. **Character rain canvas** — `#ascii-rain` behind every slide, opacity 0.22, blends in screen mode
2. **Bracket frames** — `┌─` / `─┘` corner brackets on framed content (`.as-frame`)
3. **Typewriter title** — first heading on each slide types in with blinking cursor
4. **Glyph scramble** — large numbers (`.as-scramble`) flicker random characters then resolve
5. **Phosphor green accent** — single accent color, used sparingly: labels, numbers, list bullets, divider tag

## Checklist / 生成前检查

- [ ] Every text element uses **JetBrains Mono** (or fallback monospace) / 每段文字都用等宽字体
- [ ] Background includes the `#ascii-rain` canvas + the rain JS / 背景包含字符雨 canvas 与脚本
- [ ] At most **one** typewriter title per slide / 每页最多一个打字机标题
- [ ] Stat numbers use class `.as-scramble` / 数据数字用 `.as-scramble`
- [ ] No serif fonts, no emoji, no rounded corners > 4px / 无衬线、无 emoji、圆角 ≤ 4px
- [ ] All sizes use `clamp()` / 所有尺寸用 `clamp()`
- [ ] `prefers-reduced-motion` disables rain + typewriter + scramble / 减少动画时关闭以上
